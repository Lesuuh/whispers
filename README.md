````md
# 📝 AnonBlog

A platform to **share your thoughts anonymously**.  
Built with **Next.js 15** (frontend) and **Express.js** (backend).

---

## 🚀 Features

- Post thoughts anonymously
- Browse by categories
- Comment on posts
- Responsive design for mobile & desktop
- Smooth UI animations and gradients

---

## 🏗️ Tech Stack

- **Frontend:** Next.js, Tailwind CSS, React Markdown, Axios
- **Backend:** Express.js, Node.js
- **Icons:** Lucide React

---

## 💻 Getting Started

### 1️⃣ Clone the repo

```bash
git clone https://github.com/<your-username>/<repo-name>.git
cd <repo-name>
```
````

### 2️⃣ Frontend

```bash
cd client
npm install
npm run dev
```

Runs at: `http://localhost:3000`

### 3️⃣ Backend

```bash
cd server
npm install
npm start
```

Runs at: `http://localhost:5000`

---

## ⚙️ Environment Variables

**Server (`server/.env`):**

```
PORT=5000
MONGO_URI=<your-mongo-uri>
JWT_SECRET=<your-secret>
```

**Client (`client/.env.local`):**

```
NEXT_PUBLIC_API_BASE_URL=http://localhost:5000
```
